## 1. 软件概述

### 1.1 软件名称及版本
**软件名称**：Smart QC Tools 工业级质量控制软件  
**版本号**：V1.0  
**功能模块**：雷达图分析模块（Radar Analysis Module）

### 1.2 开发背景与用途
雷达图（Radar Chart）又称蜘蛛网图，是多变量数据分析中不可或缺的可视化工具。在质量管理中，它常用于对比不同产品、班次或供应商在多个维度（如良率、成本、交期、服务等）上的综合表现。

本模块通过 **多维向量分析引擎**，支持数据自动化标准化处理，能够消除不同轴之间量纲差异，并内置了多边形面积评分与欧氏距离相似度算法，为复杂的综合评价提供科学的量化依据。

## 3. 核心功能及操作规程

### 3.1 模式一：手动编辑模式 (Manual Data Entry)
用户通过 UI 界面直接定义指标与数据。

**操作规程**：
1. **维度指标定义**：
   - 点击 **“+”** 增加维度。
   - 设定“指标名称”、该维度的“最大值 (MAX)”及“最小值 (MIN)”。
2. **数据系列定义**：
   - 点击 **“+”** 增加对比系列（如“产品 A”、“基准线”）。
   - 为每个维度输入观测值，系统将实时映射到极坐标系中。
3. **统计控制开关**：
   - **自动数据标准化**：勾选后，所有轴数据将映射至 [0, 1] 区间，适合不同量纲指标的横向对比。
   - **展示面积评分**：基于多边形面积计算综合性能分数。
   - **展示相似度**：计算其余系列与首个系列（Benchmark）的欧式距离一致性。

---

### 3.2 模式二：DSL 专家模式 (DSL Expert Mode)
**语法规范示例**：
- `Title: 综合性能分析`
- `Standardize: true`
- `Axis: 成本, 100, 10` (名称, 最大, [最小])
- `Series: 产品 A, [80, 70, 90], #3b82f6, 0.4` (名称, 数据, [颜色], [透明度])

---

### 3.3 自动化分析逻辑 (Core Analysis Logic)
本模块不仅是绘图工具，更包含深度的分析能力：
1. **多边形面积评分法**：
   公式：$Area = \frac{1}{2} \sum |x_i y_{i+1} - x_{i+1} y_i|$
   反映了对象的综合“强度”，指标越均衡，包围面积越趋近于圆，性能越稳健。
2. **欧氏距离相似度**：
   通过计算多维空间的几何距离，量化各样本与目标基准的偏离程度，适用于供应商准入与一致性评估。

## 4. 交互特性
- **自由缩放与平移**：支持鼠标滚轮对绘图区进行缩放，左键点击抓取并平移视图，方便观察复杂的多指标重叠区域。
- **整理布局**：点击顶部按钮，可使图表瞬间回归最佳初始视图（Edge-Fit 模式）。
