# VChart (VisActor) 引擎用户手册

VChart 引擎是 Smart QC Studio 中功能最强大的图表组件，它直接集成了字节跳动开源的 [VisActor VChart](https://www.visactor.io/vchart) 渲染引擎。通过该组件，您可以突破基础图表的限制，实现任何复杂的定制化数据可视化方案。

## 1. 核心优势
- **全能渲染**：支持超过 50 种基本图表类型及任意组合。
- **AI 驱动**：通过自然语言描述，AI 可自动生成复杂的 Spec 配置。
- **主题联动**：图表视觉主题（明暗模式）自动跟随 Smart QC Studio 系统设置。
- **导出能力**：支持高清 PNG 导出及 PDF 报告生成。

## 2. DSL 语法说明

VChart 的 DSL 由三部分组成：

```text
Title: [图表标题]
* ColorPalette: [ID] - 配色方案 ID（如：`tech`, `vibrant` 等）。
Font[Title]: [Size]
Font[Base]: [Size]
Font[Legend]: [Size]
Font[Axis]: [Size]
Spec: {
  [VChart 标准 JSON Spec]
}
```

### 关键字段说明
*   **Title**: 定义显示在图表顶部的标题。
*   **ColorPalette**: 设置预设的配色方案，自动映射到 Spec 的 `color` 数组。
*   **Font[...]**: 精确控制标题、正文（Tooltip 等）、图例和坐标轴的字体大小。
*   **Spec**: 必须是一个合法的 JSON 对象。它定义了数据、图表类型（type）、坐标轴（axes）、图例（legends）等所有视觉细节。

## 3. 编辑模式

### 3.1 全局配置 (Manual)
在“全局配置”标签页中，您可以直接通过 UI 控件调整：
- **图表标题**：实时修改标题文本。
- **字号控制**：分别调整标题、正文、图例、坐标轴的字体大小。
- **配色方案**：选择预设的高端配色方案（如科技蓝、工业灰等）。

### 3.2 JSON 脚本 (DSL)
在“JSON 脚本”标签页中，您可以直接编辑 DSL 源码。修改后的内容会实时同步到 UI 控件及视图中。

### 3.3 AI 智能生成 (AI Inference)
通过输入自然语言（例如：“展示去年各季度的销售额对比，并标记出最大值”），AI 会根据您的需求自动生成完整且符合规范的 VChart Spec。您也可以在描述中加入字号和颜色的要求。

## 4. 常见问题 (FAQ)

**Q: 为什么我的图表没有显示？**
A: 请检查 `Spec:` 后的 JSON 是否合法。缺少逗号、引号不匹配等都会导致解析失败。您可以将内容复制到在线 JSON 验证工具中检查。

**Q: 如何让图表在深色模式下自动变色？**
A: VChart 引擎已与 Smart QC Studio 的全局主题联动。只要确保 Spec 中没有硬编码过多的颜色，它会自动根据系统背景调整文字和坐标轴显色。

**Q: 我可以定义自己的配色吗？**
A: 您可以在 `Spec` 的 `color` 数组中定义自定义色板，或者在“全局配置”中选择我们为您优化过的配色方案。
